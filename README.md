Google Homepage Clone
This project is a clone of the Google homepage built using HTML and CSS. The goal was to practice front-end development skills by recreating Google's minimalist design, including the search bar, navigation links, buttons, and overall layout. The project focuses on clean code structure, responsive design, and attention to detail.
Technologies Used:
HTML5
CSS3
Features:
Responsive layout
Google-inspired UI
Clean and semantic HTML structure
Modern CSS styling
Feel free to explore, fork, and contribute!
